# Time-Compass
Display real/current:
- analog clock in 24-hours, 12-hours, 60-minutes, 60-seconds
- digital clock HH:MM:SS
- date (2025-10-08)
- day (Sunday, Monday, Tuesday, ...)
- in Arab, Malay, English
